Politique de Confidentialité

La présente politique explique comment 7ensemble.ch collecte, utilise et protège les données personnelles des utilisateurs.

1. Responsable du traitement

Le responsable du traitement des données est :

Christophe Rossier, pour le projet 7ensemble.ch

Localisation : 1926 Fully (VS), Suisse

Contact : mission@7ensemble.ch

2. Données collectées

Les données peuvent être collectées lorsque vous :

  • remplissez un formulaire (inscription / contact)
  • envoyez un email
  • naviguez sur le site

Types de données possibles :

  • prénom / nom (si fourni)
  • email
  • contenu du message
  • données techniques (adresse IP, navigateur, pages consultées, cookies)

3. Finalités

Les données sont utilisées uniquement pour :

  • répondre aux messages
  • gérer les inscriptions et les demandes
  • améliorer l'expérience utilisateur
  • assurer la sécurité du site

4. Base légale (RGPD & Suisse)

Le traitement des données repose sur :

  • Consentement : lorsque vous remplissez un formulaire
  • Intérêt légitime : pour assurer le bon fonctionnement du site
  • Exécution d'un contrat : si vous utilisez un service proposé

5. Conservation des données

Les données sont conservées le temps nécessaire pour atteindre les finalités prévues.

En général :

  • Messages de contact : jusqu'à résolution de la demande
  • Données d'inscription : tant que le compte est actif
  • Données techniques : durée limitée (logs serveur)

6. Partage des données

Aucune donnée n'est vendue ou cédée à des tiers à des fins commerciales.

Les données peuvent être partagées uniquement :

  • Avec l'hébergeur (Infomaniak) dans le cadre technique
  • Si la loi l'exige

7. Vos droits (RGPD & LPD suisse)

Vous disposez des droits suivants :

  • Accès : obtenir une copie de vos données
  • Rectification : corriger vos informations
  • Suppression : demander l'effacement
  • Opposition : refuser certains traitements
  • Portabilité : récupérer vos données dans un format lisible

Pour exercer ces droits, contactez : mission@7ensemble.ch

8. Cookies

Le site peut utiliser des cookies pour :

  • Améliorer la navigation
  • Mémoriser vos préférences
  • Analyser l'utilisation du site

Vous pouvez désactiver les cookies dans les paramètres de votre navigateur.

9. Sécurité

Des mesures techniques et organisationnelles sont mises en place pour protéger vos données contre tout accès non autorisé, perte ou altération.

10. Modifications

Cette politique peut être modifiée à tout moment.

La version en vigueur est celle publiée sur le site.
